#  Online Bookstore SQL Project

## Overview

This project is a PostgreSQL database project based on an **Online Bookstore Management System**. It demonstrates database design, data import, and SQL query writing using real-world business scenarios.

The project includes creating relational tables, importing CSV data, and solving beginner to intermediate SQL problems involving filtering, aggregation, joins, grouping, sorting, and business analysis.

## Database Schema

The database contains three related tables:

* **Books**
* **Customers**
* **Orders**

Relationships are established using **Primary Keys** and **Foreign Keys** to maintain data integrity.

## Features

* Database creation
* Table creation with constraints
* Primary & Foreign Key relationships
* CSV data import using `COPY`
* Data retrieval queries
* Aggregate functions
* GROUP BY and HAVING
* ORDER BY and LIMIT
* Business analysis queries
* Inventory analysis
* Customer analysis
* Sales and revenue analysis

## Business Questions Solved

Some of the SQL problems included in this project are:

* Retrieve books by genre
* Find books published after a specific year
* Identify customers from a particular country
* Calculate total revenue
* Find the most expensive book
* Retrieve books with the lowest stock
* Analyze books sold by genre
* Find the most frequently ordered book
* Identify top customers by spending
* Calculate remaining inventory after sales
